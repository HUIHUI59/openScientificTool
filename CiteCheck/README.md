# CiteCheck · 参考文献打假体检器

> 把你的参考文献清单（`.bib` 或纯文本）逐条拿 **CrossRef 官方数据库**过一遍：
> DOI 是真是假、题目/作者/年份对不对得上、缺了哪些字段、格式统不统一——
> 出一份 ✅/⚠️/🔴 体检报告。**只体检，不改你任何文件，更不帮你找文献。**

```
references.bib  ──►  references_体检报告.txt
   [1] ✅ 通过 | Orben & Przybylski (2019) ...   DOI 验真通过, 题目/年份与官方记录一致
   [2] 🔴 高危 | Smith (2021) ...                DOI 返回 404 —— 查无此号, 疑似编造
   [3] 🔴 高危 | Baumeister & Leary (1992) ...   DOI 真实但年份对不上(条目 1992 / 官方 1995)
   [4] ⚠️ 待核 | 王XX (2020) 某中文期刊 ...       无 DOI 无法核验(不等于编造), 请到知网自查
```

---

## 🔒 铁律（先读这四条，这就是产品边界）

1. **绝不代找文献。** 它不会"帮你的观点配几条引用"——只体检你已经有的条目。
2. **报告不是论文内容，不可粘进论文。** 输出只有体检结论和修复建议。
3. **不改你的任何文件。** 输入只读，报告另存新文件；哪条要改、怎么改，由你自己动手。
4. **"无法核验" ≠ "编造"。** 中文知网/万方文献大多没有 DOI、不在 CrossRef 库内，它会如实标"无法核验"并告诉你去哪自查，绝不武断判假。

---

## 这是什么 / 给谁用

AI 编造不存在的文献（幻觉引用）已经是公开的老问题：题目像模像样、作者真实存在、DOI 格式也正确——但那篇论文根本不存在。混进参考文献列表里，投稿前不查出来就很难看。

**为什么不直接问 GPT"这条文献是真的吗"？** 因为它自己就是会编文献的那位。
**CiteCheck 问的是 CrossRef 官方数据库**（全球学术出版物的 DOI 注册机构）：404 就是 404，查得到就是查得到，报告里的每个 DOI 你都可以自己去 doi.org 再验一遍，不必信它的一面之词。

适合：社科 / 人文 / 理工科的硕博、科研人员——凡是「参考文献列表交出去之前，想逐条过一遍真伪和格式」的场景。**不需要你会写代码。**

## 它体检哪几样

| 体检项 | 怎么查 | 结果 |
|---|---|---|
| **DOI 真伪** | `api.crossref.org/works/{doi}`，404 = 查无此号 | 🔴 高危 |
| **题录一致性** | DOI 真实时，比对官方题目/第一作者/年份 | 对不上 → 🔴（附官方值） |
| **无 DOI 条目** | 按题录模糊检索，高相似命中给出比对结果 | 查不到 → ⚠️ 无法核验（≠编造） |
| **缺项** | 期刊文章缺卷期/页码、图书缺出版社等 | ⚠️ 列缺什么 |
| **格式统一** | 全半角标点混用、作者写法混用、DOI 写法混用 | 清单级问题列表 |

## 纯文本也能喂（不是只吃 .bib）

很多人用 Word + Zotero，手里未必有 `.bib`。把参考文献列表整段复制存成 `.txt`（一条一行，或条与条之间空一行）直接喂给它就行。注：纯文本没有字段结构，题目/作者靠 DOI 和模糊检索反向比对，个别条目的判断会保守一些（保守 = 宁标"待核"也不乱判）。

## 中文文献的边界（如实说）

知网/万方收录的中文文献**大多没有 DOI、也不在 CrossRef**——这类条目真伪核不了，报告会如实标"⚠️ 无法核验 + 原因"，只对它做格式与缺项体检，并提示你**到知网/万方按题名检索自查**。别的工具装作能查，本工具选择说实话。

## 注意事项

- **运行时需要联网**（查 CrossRef 官方库）。断网时条目会标"服务暂不可用, 未核验"，不硬判。
- 免费 API 礼貌使用：每条间隔 1 秒，100 条大约要跑两三分钟，属正常速度。
- 隐私：不上传你的文件，只把 DOI / 题录字段发给 CrossRef 查询。
- 报告给出的"官方值"仅供你自己核对修改；**最终改不改、怎么改，永远是你的决定**（人在回路）。
